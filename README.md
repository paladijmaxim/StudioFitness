# Fitness Studio 

## Описание проекта
Веб-приложение для управления фитнес-центром с информацией о занятиях, тренерах и клубах. Проект реализован на Django с современным интерфейсом и базовыми CRUD операциями.

## Технологический стек

- **Backend**: Django 5.2.7
- **Frontend**: HTML5, CSS3
- **База данных**: SQLite
- **Стили**: CSS Grid, Flexbox
- **Язык программирования**: Python 3.13

### Для модели Event реализованы:
- Create - создание новых событий
- Read - просмотр списка и детальной информации
- Update - редактирование существующих событий
- Delete - удаление событий с подтверждением
